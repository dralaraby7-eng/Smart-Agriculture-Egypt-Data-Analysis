# 🌱 Egypt Agriculture Analytics - FAOSTAT

## 📌 Project Overview

This project analyzes **25 years of agricultural data in Egypt** using FAOSTAT datasets, focusing on:

* 🌾 Crop production & yield
* 🌧 Rainfall & temperature impact
* 🌍 CO2 emissions
* 🌱 Fertilizer usage
* 👥 Population growth
* 🧑‍🌾 Arable land

The goal is to provide **data-driven insights for sustainable agriculture and food security**.

---

## 🎯 Objectives

* Analyze agricultural trends in Egypt
* Measure efficiency of water and fertilizer usage
* Study climate impact on crop production
* Evaluate sustainability indicators
* Support decision-making using data

---

## 🛠 Tools & Technologies

* Power BI (Dashboard & Data Modeling)
* Python (Data Analysis & Prediction)
* Pandas, Scikit-learn
* FAOSTAT Dataset

---

## 📊 Dashboard Features
